Fog Project

Solução de clonagem de imagens de sistemas operativos via rede.

Sobre o Projeto

Descrição

O FOG Project é uma ferramenta open-source baseada em Linux que permite criar e restaurar imagens completas de discos rígidos de computadores remotamente através da rede (usando PXE boot). Este projeto focou-se na otimização do tempo de configuração de novos equipamentos no departamento de TI.

Vantagens & Implementação

  • Configuração de um servidor PXE para boot de rede.
  • Criação de "Master Images" (Imagens modelo) para implementação rápida e em massa de sistemas operativos.
  • Redução drástica do tempo necessário para formatar e configurar novos postos de trabalho.
  • Gestão remota e agendamento de tarefas de imagem.
