Mini-Servidor com Proxmox VE

Plataforma de virtualização para gestão de ambientes de TI.

Sobre o Projeto

Descrição

Implementação de um mini-servidor baseado em Proxmox Virtual Environment (VE). Esta solução empresarial open-source permitiu gerir máquinas virtuais e containers isolados a partir de uma interface web única, centralizando recursos e poupando custos de hardware.

Principais Tecnologias e Configurações

  • Instalação e configuração do hipervisor bare-metal Proxmox VE.
  • Criação de Máquinas Virtuais (VMs) Linux/Windows para serviços internos.
  • Implementação de containers LXC para aplicações leves (ex: Pi-hole, web servers).
  • Configuração de backups automáticos (Snapshots).
  • Gestão de rede virtual e alocação dinâmica de recursos (RAM e CPU).
